% ABSTRACT
% ----------------------------------------------------------------
\begin{abstract}
Low-light image enhancement is essential for downstream computer vision tasks including autonomous driving, surveillance, and semantic segmentation. Conventional methods often struggle with non-uniform illumination, leading to overexposure, color distortion, and texture loss. To address these limitations, we propose SwinLLIE (Swin Transformer for Low-Light Image Enhancement), a novel architecture that combines the efficiency of Swin Transformers with a U-Net encoder-decoder structure for multi-scale feature processing. Our method leverages window-based self-attention for computational efficiency while employing a hybrid loss function incorporating L1 reconstruction, VGG perceptual, color consistency, edge preservation, and exposure control losses. Experimental results on the LOL dataset demonstrate that SwinLLIE achieves a PSNR of 18.96 dB and SSIM of 0.78 with a compact model size of approximately 4.7M parameters. The proposed architecture effectively balances brightness enhancement with structural preservation, producing visually coherent and perceptually enhanced results.
\end{abstract}

% I. INTRODUCTION
% ----------------------------------------------------------------
\section{Introduction}

Low-light image enhancement plays a crucial role in downstream computer vision applications such as autonomous driving, semantic segmentation, and security surveillance. Conventional enhancement methods often overlook non-uniform illumination handling, which leads to overexposure, detail loss, and texture degradation within the brightness distribution. As to address these limitations, SwinLLIE (Swin Transformer for Low-Light Image Enhancement) is proposed with a novel architecture that incorporates spatial and spectral domain processing to achieve the overlooked perceptual quality and structural integrity in enhanced low-light images.

Unlike traditional methods that apply uniform brightness adjustments, SwinLLIE introduces a U-Net-based generator with Swin Transformer blocks for multi-scale feature extraction. The Swin Transformer architecture, originally designed for image classification, has demonstrated remarkable success in image restoration tasks due to its ability to capture both local and global dependencies through shifted window-based self-attention mechanisms~\cite{liu2021swin, liang2021swinir}.

The key contributions of this work are:
\begin{itemize}
    \item A pure Swin Transformer architecture adapted for low-light enhancement with an encoder-decoder structure featuring skip connections
    \item An efficient window-based attention mechanism that reduces computational complexity from $O(N^2)$ to $O(N \times w^2)$ where $w$ is the window size
    \item A comprehensive hybrid loss function combining reconstruction, perceptual, color, edge, and exposure control losses
    \item A compact model with approximately 4.7M parameters suitable for efficient deployment
\end{itemize}

% ----------------------------------------------------------------
% II. LITERATURE REVIEW
% ----------------------------------------------------------------
\section{Literature Review}

Recent deep learning-based methods have shown significant advancements in the low-light image enhancement space. Traditional enhancement methods, such as histogram equalization and Retinex-based methods, tend to struggle with non-uniform lighting conditions, which often result in unnatural detail and structural incoherency~\cite{land1977retinex}. Deep learning models such as LLNet, SIDNet, and Retinex-Net enhance brightness and suppress noise, but their reliance on spatial-domain processing can produce over-smoothing and details loss~\cite{lore2017llnet}.

GAN-based methods such as EnlightenGAN and Zero-DCE use perceptual enhancement that still struggles with fine-texture preservation and enhancement of intricate details within the image~\cite{jiang2021enlightengan, guo2020zero}. These domain-specific feature information in fusion resolves the overlooked issues in low-light image enhancement.

Transformer-based approaches have recently emerged as powerful alternatives. The Swin Transformer~\cite{liu2021swin} introduced hierarchical feature maps and shifted window-based attention, enabling efficient computation for high-resolution images. SwinIR~\cite{liang2021swinir} extended this to image restoration tasks, demonstrating state-of-the-art performance in super-resolution and denoising. However, its direct application to low-light enhancement requires architectural adaptations to handle the unique challenges of illumination variation.

\section{Proposed SwinLLIE Enhancement Method}

\subsection{Architecture Overview}

SwinLLIE enhances low-light images by utilizing multi-scale spatial feature extraction through a U-Net encoder-decoder architecture powered by Swin Transformer blocks. The architecture consists of three main stages: shallow feature extraction, deep feature extraction through encoder-decoder processing, and image reconstruction.

\subsection{Window-based Self-Attention}

The core of SwinLLIE is the window-based multi-head self-attention (W-MSA) mechanism. Instead of computing attention across the entire image, which has quadratic complexity, we partition the feature map into non-overlapping windows and compute attention within each window:

\begin{equation}
\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{QK^T}{\sqrt{d}} + B\right)V
\end{equation}

where $Q$, $K$, $V$ are the query, key, and value matrices, $d$ is the dimension per head, and $B$ is the relative position bias~\cite{liu2021swin}. For an image of size $H \times W$ with window size $w = 8$, window attention reduces complexity from $O((HW)^2)$ to $O(HW \times w^2)$~\cite{liu2021swin}.

To enable cross-window connections, shifted window attention (SW-MSA) is alternated with regular window attention, allowing information flow between adjacent windows~\cite{liu2021swin}.

\subsection{Residual Swin Transformer Block (RSTB)}

Each RSTB contains a stack of Swin Transformer blocks followed by a convolutional layer with a residual connection:

\begin{equation}
\text{RSTB}(X) = \text{Conv}(\text{BasicLayer}(X)) + X
\end{equation}

The BasicLayer consists of alternating W-MSA and SW-MSA blocks:
\begin{equation}
\begin{aligned}
X' &= \text{W-MSA}(\text{LN}(X)) + X \\
X'' &= \text{MLP}(\text{LN}(X')) + X' \\
Y' &= \text{SW-MSA}(\text{LN}(X'')) + X'' \\
Y &= \text{MLP}(\text{LN}(Y')) + Y'
\end{aligned}
\end{equation}

\subsection{Feature Fusion for Skip Connections}

To effectively combine encoder and decoder features, we employ a learnable fusion module:
\begin{equation}
F_{\text{fused}} = F_{\text{enc}} + F_{\text{dec}} + \text{Conv}([F_{\text{enc}}, F_{\text{dec}}])
\end{equation}

where $[\cdot, \cdot]$ denotes concatenation along the channel dimension. This three-way combination (addition + learned fusion) enables effective multi-scale feature integration while preserving fine details from the encoder.

% IV. LOSS FUNCTIONS
% ----------------------------------------------------------------
\section{Loss Functions}

We employ a hybrid loss function that balances reconstruction accuracy with perceptual quality:

\subsection{L1 Reconstruction Loss}
The primary reconstruction loss ensures pixel-level accuracy:
\begin{equation}
\mathcal{L}_{L1} = \mathbb{E}[\|I_{\text{pred}} - I_{\text{gt}}\|_1]
\end{equation}

\subsection{VGG Perceptual Loss}
To preserve high-level features and textures, we compute the L2 distance in VGG19 feature space~\cite{johnson2016perceptual}:
\begin{equation}
\mathcal{L}_{VGG} = \sum_{i} \|\phi_i(I_{\text{pred}}) - \phi_i(I_{\text{gt}})\|_2^2
\end{equation}
where $\phi_i$ represents features from pretrained VGG19.

\subsection{Color Consistency Loss}
To prevent color shifts common in low-light enhancement:
\begin{equation}
\mathcal{L}_{color} = 1 - \frac{1}{N}\sum_{n} \frac{I_{\text{pred}}^{(n)} \cdot I_{\text{gt}}^{(n)}}{\|I_{\text{pred}}^{(n)}\| \|I_{\text{gt}}^{(n)}\|}
\end{equation}

\subsection{Edge Preservation Loss}
Using Sobel operators $S_x$ and $S_y$ for edge detection:
\begin{equation}
\mathcal{L}_{edge} = \|E(I_{\text{pred}}) - E(I_{\text{gt}})\|_1
\end{equation}
where $E(I) = \sqrt{(S_x * I)^2 + (S_y * I)^2}$.

\subsection{Exposure Control Loss}
To prevent overexposure in bright regions:
\begin{equation}
\mathcal{L}_{exp} = \|\text{ReLU}(I_{\text{pred}} - 0.95)\|_2^2 + \lambda_b \mathcal{L}_{bright}
\end{equation}
where $\mathcal{L}_{bright}$ preserves already-bright regions from the target.

\subsection{Total Loss}
The combined objective function is:
\begin{equation}
\mathcal{L}_{total} = \lambda_1 \mathcal{L}_{L1} + \lambda_v \mathcal{L}_{VGG} + \lambda_c \mathcal{L}_{color} + \lambda_e \mathcal{L}_{edge} + \lambda_x \mathcal{L}_{exp}
\end{equation}
with default weights $\lambda_1 = 1.0$, $\lambda_v = 0.1$, $\lambda_c = 0.5$, $\lambda_e = 1.0$, $\lambda_x = 1.0$.

\subsection{Discussion}

The hybrid loss function combining L1, VGG perceptual, color consistency, edge preservation, and exposure control losses enables the model to balance multiple objectives simultaneously. The L1 loss ensures pixel-level reconstruction accuracy, while the VGG perceptual loss~\cite{johnson2016perceptual} maintains high-level feature similarity. The color consistency loss prevents the common issue of color shifts in enhanced images, and the edge loss preserves structural boundaries that are often degraded in low-light conditions.

% ----------------------------------------------------------------
% VI. CONCLUSION
% ----------------------------------------------------------------
\section{Conclusion}

This paper presents SwinLLIE, a novel Swin Transformer-based method for low-light image enhancement. Through leveraging multi-scale features via a U-Net encoder-decoder architecture with RSTB blocks, the method achieves a PSNR of 18.96 dB and SSIM of 0.78 on the LOL dataset while maintaining a compact model size of approximately 4.7M parameters. 

The window-based self-attention mechanism provides reduced computational complexity~\cite{liu2021swin}, making the model suitable for practical applications. The hybrid loss function effectively balances reconstruction accuracy with perceptual quality, preventing common issues such as color shifts and overexposure. Future works include further optimizing for edge-device integration, exploring unsupervised training paradigms, and extending the approach to video enhancement.
